Introduction:

This is a backend platform for a Pizza Store, you can have 3 types of Users (Customer, Cashier, Admin)

Customer and Cashier have the ability to create orders (Limited to Pizza in this demo)

Admin can create reports on all the data stored in said database, in this demo a report for all sales can be made that takes into account all sales of pizzas


Future feature:

	-Admin can create sales found by each Cashier.
	-Admin can create sales reports based on the date of order.
